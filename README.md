# AiSandTable

AiSandTable 是一款专为CEO、CIO、企业战略顾问打造的，基于大模型人工智能技术的企业竞争沙盘系统。我们聚焦企业竞争情报分析，通过AI技术自动化采集、分析、汇总核心竞品数据，为企业战略决策提供精准、高效、可视化的数据支撑。

系统采用企业内部Wiki模式存储竞争数据，完整保留数据更新历史，支持跨时间维度比对竞争对手的战略变化、产品迭代、市场布局等核心信息，让企业决策层精准掌握竞争格局，抢占市场先机。

作为完全开源的企业级系统，AiSandTable支持私有化部署，所有竞争数据安全存储在企业内部，无数据外泄风险，完全符合企业数据安全合规要求。
